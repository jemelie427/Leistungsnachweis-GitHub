## Projektidee
Wir untersuchen, bei welchen Länder von 1990 bis 2023 die Säuglingssterblichkeit (Kinder <1 Jahr) am meisten zugenommen und wo sie meisten abgenommen hat. Zudem recherchieren wir mögliche Gründe dafür.
